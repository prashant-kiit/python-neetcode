# Python Notes: Decorators, Iterators, Generators, `*args`, `**kwargs`

## 1. Decorators

**Definition:**
A decorator is a function that wraps another function to add or modify its behavior without changing the original function.

**Syntax**

```python
def log(func):
    def wrapper():
        print("Start")
        func()
    return wrapper

@log
def hello():
    print("Hello")
```

**Usage**

* Logging
* Authentication/Authorization
* Caching
* Timing function execution

---

## 2. Iterators

**Definition:**
An iterator is an object that returns one item at a time using the `__iter__()` and `__next__()` methods.

**Example**

```python
it = iter([10, 20, 30])

print(next(it))  # 10
print(next(it))  # 20
```

**Usage**

* Traverse collections one element at a time.
* Memory-efficient iteration over large datasets.

---

## 3. Generators

**Definition:**
A generator is a function that produces values lazily using the `yield` keyword instead of returning all values at once.

**Example**

```python
def count():
    yield 1
    yield 2

for i in count():
    print(i)
```

**Usage**

* Processing large files.
* Infinite sequences.
* Memory-efficient data pipelines.

## 4. Function Arguments (`*args`, `**kwargs`)

**Definition:**

* `*args` collects any number of **positional arguments** into a **tuple**.
* `**kwargs` collects any number of **keyword arguments** into a **dictionary**.

### Example

```python
def show(*args, **kwargs):
    print(args)
    print(kwargs)

show(10, 20, 30, name="John", age=25)
```

**Output**

```python
(10, 20, 30)
{'name': 'John', 'age': 25}
```

### How they work

#### `*args`

* Accepts extra positional arguments.
* Stored as a **tuple**.

```python
def add(*nums):
    print(nums)

add(1, 2, 3)
# (1, 2, 3)
```

#### `**kwargs`

* Accepts extra keyword arguments.
* Stored as a **dictionary**.

```python
def info(**data):
    print(data)

info(name="John", age=25)
# {'name': 'John', 'age': 25}
```

### Using both together

```python
def greet(greeting, *names, **options):
    for name in names:
        print(f"{greeting} {name}")
    print(options)

greet("Hi", "Alice", "Bob", lang="en", emoji="👋")
```

**Output**

```python
Hi Alice
Hi Bob
{'lang': 'en', 'emoji': '👋'}
```

### Common Uses

* Flexible function APIs.
* Wrapper functions and decorators.
* Passing arguments to other functions.
* Handling optional parameters.

> **Note:** `args` and `kwargs` are only naming conventions. The `*` and `**` operators give them their meaning, so names like `*numbers` and `**config` work the same way.
