Python Slicing

Syntax:
sequence[start:stop:step]

Rules:
• start → Inclusive
• stop → Exclusive
• step → Default = 1
• step > 0 → Left → Right
• step < 0 → Right → Left

Example:
text = "Python"

Indices:
  P  y  t  h  o  n
  0  1  2  3  4  5

Examples:
text[:]       → 'Python'   # Copy / entire sequence
text[:4]      → 'Pyth'     # First 4 characters
text[1:]      → 'ython'    # From index 1 to end
text[1:4]     → 'yth'      # Index 1 to 3
text[::-1]    → 'nohtyP'   # Reverse
text[::2]     → 'Pto'      # Every 2nd character
text[::-2]    → 'nhy'      # Every 2nd character in reverse
text[4:1:1]   → ''         # Empty (can't move forward)
text[4:1:-1]  → 'oht'      # Move backward

Memory Tricks:
[:]      → Copy everything
[:n]     → First n elements
[n:]     → From index n to end
[a:b]    → a (inclusive) to b (exclusive)
[a:b:s]  → Slice with step s
[::-1]   → Reverse
[::2]    → Every 2nd element
[::-2]   → Reverse every 2nd element
